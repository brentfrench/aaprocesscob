# AA Process Coach

This folder contains a static chatbot-style guide built from `Revised AA workflow v1.xlsx` and the Word forms in the source folder. Each form action downloads the matching `.docx` form.

## Open locally

Open `index.html` in a browser.

If your browser blocks direct Word-form links from a local file, run a small local server from this folder:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Included forms

- `APC Academic Program Change Form.docx`
- `PRNACMCP Preliminary Review for New Academic Minor or Certificate Program Form.docx`
- `PRNP Preliminary Review for New Program Form.docx`
- `UNAP UAA New Program Application Form.docx`
- `WNCCP WCBT New or Changed Course Proposal Form.docx`
- `WNCPP New or Changed Program Form.docx`
